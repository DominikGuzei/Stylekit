Quick Start

On the left side you see the Css-Panel which holds all the styles that get added to the currently viewed website. You can edit the rules in realtime and you will see the changes immediatly in the WebView. To see how the styles affect other sites simply click on any link or enter the url directly into the Url-Bar at the top. You can also edit the HTML source of any website in the panel below this WebView. Drag the splitter vertically to the top to reveil the code of the website.

Realtime Projects

As web designer it is a tedious task to write css having no clue how it will look like in the ever growing and diversing browser landscape. StyleKit helps you out with realtime cross-browser projects based on URLs. Simply click the 'save' icon in the menu bar and you are directed to your newly created project url. Copy and Paste it in any other browser and enable 'realtime auto reloading' in the menu. The changes you make in one browser are now visible in the other browsers.

Dev-Candy

The two icons to the right of the Url-Bar provide access to utility features like the well known Firebug-Lite, a debugging tool that helps figuring out which rules affect the html markup and override them in the Css-Panel. You will find the small firebug icon in the right bottom corner of the WebView. This enables you to combine well known features with the new possibilities provided by StyleKit. The last Button gives you the chance to download your current project's css as a file.
